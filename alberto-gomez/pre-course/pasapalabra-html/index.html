PASAPALABRA

El jugador tiene 150 segundos para acertar todas las palabras que se esconden en cada una de las definiciones.
Si no conoce la palabra, el jugador puede decir 'PASAPALABRA', y pasaría a la siguiente letra.
El tiempo se detendrá cuando el jugador falle una palabra.
El juego se terminará cuando el jugador haya contestado a todas las definiciones o cuando se agoten los 150 segundos. ¡Suerte!
